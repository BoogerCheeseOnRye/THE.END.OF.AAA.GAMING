# The End of AAA Gaming

A single-file browser roguelike neon FPS that mocks everything AAA gaming stands for.

**No install. No download. Just open the HTML and play.**

## About the Game

Welcome to **The End of AAA Gaming** — a violent, psychedelic, zero-pause hardcore arena shooter built entirely in one HTML file.

You are a neon-soaked warrior trapped in an endless procedural voxel world. There is no story. There is no pause. There is only **TO VALHALLA**.

- Procedural infinite terrain with pulsing rainbow blocks
- 9 stackable powerups (BONK OVERLORD, ZOOM GREMLIN, etc.)
- 13 permanent meta upgrades in the VOID ARCHIVES
- 10 achievements with 10 tiers each in the HALL OF VALOR
- Floating crit damage numbers with !!!
- Blood drips on the title screen
- FauxPVP mode and STICKSHIFT hardcore toggle

The game is a love letter to old-school browser games and a middle finger to modern AAA bloat.

**Goal:** Survive as long as possible, rack up kills, level up, earn meta currency, and become the ultimate legend in the Hall of Valor.

## How to Play

### Getting Started
1. Save the game as `the-end-of-aaa-gaming.html`
2. Double-click the file (or drag into Chrome/Firefox)
3. Click **TO VALHALLA**
4. Click the screen once to lock the mouse

### Controls
- **WASD** — Move
- **Shift** — Sprint (drains stamina)
- **Ctrl** — Crouch (hold while landing for extra slide)
- **F** — Jump
- **Mouse** — Look
- **Left Click** — Attack (hold in normal mode, auto in STICKSHIFT hardcore)
- **1 / 2** — Switch weapons (UNARMED or NEON ROD)
- **V** — Toggle third-person view
- **Q** — Give up and sacrifice honor (ends run)
- **ESC** — Release mouse (only works outside level-up)

### Progression
- Kill enemies → collect XP orbs
- Level up → choose 1 of 3 random upgrades
- Die or press Q → earn VOID SHARDS / NEON ESSENCE / ABYSS CRYSTALS
- Spend currency in **VOID ARCHIVES** for permanent stat boosts
- Check **HALL OF VALOR** for achievement progress and rewards

### Tips
- Shield always absorbs damage first
- Ranged enemy shots fizzle at 10 units
- Powerups stack and last longer when you collect duplicates
- Tips appear randomly (toggle them off in the menu)
- The first 3 XP orbs level you up — don’t waste them

### Save System
- Auto-saves every 10 seconds
- Click the 💾 icon on the title screen to export your save
- Use the hidden file input (or drag a save file) to import

### Achievements & Meta
Every run contributes to 10 different achievement lines (10 tiers each). Completing tiers gives bonus VOID SHARDS. Meta upgrades are permanent and affect every future run.

## Technical Notes

- Best in Chrome or Firefox (full-screen recommended — press F11)
