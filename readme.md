# The End of AAA Gaming

**Single-file neon roguelike FPS.** No bloat. No patches. Just pure chaotic death.

A violent, psychedelic, zero-pause hardcore arena shooter built entirely in one HTML file. You are a neon-soaked warrior trapped in an endless procedural voxel world where the terrain itself ripples outward in beautiful, clashing waves.

## Get Started

```bash
npm install
npm run dev
```

## Build for Production

```bash
npm run build
```

## Play

- **TO VALHALLA** — Start a run
- **VOID ARCHIVES** — Permanent meta upgrades (carry across runs)
- **HALL OF VALOR** — Achievements and leaderboards
- **SETTINGS** — Controls, graphics, audio

## Features

- 9 stackable powerups (BONK OVERLORD, ZOOM GREMLIN, LOWGRAV JESTER, etc.)
- 13 permanent meta upgrades
- 10 achievements with 10 tiers each
- Dynamic ripple-wave terrain
- Gamepad support (Xbox / any gamepad)
- FauxPVP multiplayer mode
- STICKSHIFT hardcore toggle

## Controls

**Keyboard + Mouse**
- **WASD** — Move
- **Shift** — Sprint
- **F** — Jump
- **Mouse** — Look
- **Left Click** — Attack
- **Q** — Give up and sacrifice honor
- **ESC** — Release mouse

**Xbox / Gamepad**
- **Left Stick** — Move
- **Right Stick** — Look
- **A** — Jump
- **RT** — Attack
